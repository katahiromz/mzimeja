(Japanese, Shift_JIS)

■「MZ-IME日本語入力」by 片山博文MZ

これは何？
----------

これは、片山博文MZによる、「MZ-IME日本語入力」(mzimeja)です。
このプログラムは、Windows 2000用の日本語IMEです。

動作環境
--------

 * 日本語 Windows 2000
 * 日本語 Windows XP
 * 日本語 Windows Vista
 * 日本語 Windows Server 2003
 * 日本語 Windows 7 

MZ-IMEの特徴
------------

 * インターネットに接続しません。
 * 学習しません。
 * そんなに賢くありません。
 * 著作権フリーです。

インストール方法
----------------

次の手順でMZ-IMEをインストールできます。

(1) アプリケーションをインストールする。

(1.1) インストーラー「mzimeja-X.X-setup.exe」(X.Xはバージョン番号) を
　　　ダブルクリックで開いて下さい。
(1.2) インストーラーの画面の指示に従ってインストールして下さい。

(2) 入力ロケールを切り替える。

(2.1) スタートメニューの「設定」→「コントロールパネル」を順番に
　　　クリックして下さい。
　　　「コントロールパネル」ウィンドウが開かれます。
(2.2) 「コントロールパネル」ウィンドウの「地域のオプション」を
　　　ダブルクリックして下さい。
　　　「地域のオプション」ダイアログが開かれます。
(2.3) 「地域のオプション」ダイアログの「入力ロケール」タブを
　　　クリックして下さい。
(2.4) 「日本語 (MZ-IME)」を左クリックで選択した後で、
　　　「既定値に設定(S)」ボタンをクリックして、「OK」ボタンを
　　　クリックして下さい。
　　　MZ-IMEが既定のIMEになります。

(3) Windowsを再起動して下さい。

アンインストール方法
--------------------

次の手順に従ってアンインストールできます。

(1) MZ-IMEの入力ロケールを削除する。

(1.1) スタートメニューの「設定」→「コントロールパネル」を順番に
　　　クリックして下さい。
　　　「コントロールパネル」ウィンドウが開かれます。
(1.2) 「コントロールパネル」ウィンドウの「地域のオプション」を
　　　ダブルクリックして下さい。
　　　「地域のオプション」ダイアログが開かれます。
(1.3) 「地域のオプション」ダイアログの「入力ロケール」タブを
　　　クリックして下さい。
(1.4) 「日本語 (MZ-IME)」を左クリックで選択した後で、
　　　「削除(R)」ボタンをクリックして、「OK」ボタンを
　　　クリックして下さい。

(2) アプリケーションを削除する。

(2.1) 「コントロールパネル」から「アプリケーションの追加と削除」を
　　　ダブルクリックして下さい。
　　　「アプリケーションの追加と削除」ダイアログが開かれます。
(2.2) 「アプリケーションの追加と削除」ダイアログから
　　　「MZ-IME日本語入力」を左クリックして選択し、
　　　「変更/削除(C)」ボタンをクリックして下さい。
　　　「MZ-IME日本語入力 とその関連コンポーネントをすべて
　　　 削除します。よろしいですか？」と表示されます。
(2.3) 「はい(Y)」ボタンをクリックすると、アンインストールされます。

使い方
------

操作方法は、普通のIMEとほとんど同じです。
ただ、かな漢字変換があまり賢くないので、変換がうまくいかないときは、
文節の長さを[Shift]+[←]と[Shift]+[→]で調節する必要があります。
辞書にない単語は、変換ができないことがあります。

注意事項
--------

まだ地名と人名の変換には対応していません。
