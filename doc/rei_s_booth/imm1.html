IME Properties

Occasionaly you may want to know the current IME's filename or capability to interact with it.
You can easily retrieve the IME properties, such as filename, description, supported conversion/sentence modes, whether unicode is supported or not etc., using ImmGetIMEFileName, ImmGetDescription and ImmGetProperty functions.

These functions take the handle to the keyboard layout (hKL) as a parameter. Normally the current hKL which holds the IME selected for the current input context (window) is passed to them. It can be obtained by calling GetKeyboardLayout(0).

To ensure the proper result with every IME, the hKL should be retrieved exactly before the call of the function. For example, it is not recommended to share the hKL stored in a global variable with some functions. Note that the keyboard layout can be changed on the fly during the same process by the user's operation.
