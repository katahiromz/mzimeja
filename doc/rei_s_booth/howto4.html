How to Activate Software Keyboard

If you are using Microsoft IME98, you can display its "IME Pad" by using ImmSetConversionStatus. The following code invokes IME Pad selecting software keyboard applet.

There may be other IMEs which support software keyboard correctly, but sorry that I don't know. And I'm not sure whether Chinese IMEs support this interface or not.

 

HIMC hIMC;
DWORD dwConvMode, dwSentMode;

hIMC = ImmGetContext(hWnd);
ImmGetConversionStatus(hIMC, &dwConvMode, &dwSentMode);

dwConvMode &= ~IME_CMODE_CHARCODE;
dwConvMode |= IME_CMODE_SOFTKBD;
ImmSetConversionStatus(hIMC, dwConvMode, dwSentMode);

ImmReleaseContext(hWnd, hIMC);
