IMEDic Sample

IMEDic sample is a tiny dictionary editor for Win95/NT4 IMEs. You can view the words in the IME's user dictionary on a List View Control, register a new word, edit and delete the selected word. In addition, the word register dialog of IMEDic will show you how to change the IME's conversion mode.

This program is designed for FE version of Win95/NT4 with Japanese IMEs. Tested under NT4J SP3 with MS-IME 97 and MS-IME 98.

Download IMEDic Sample : imedic.zip, 24kb

IMEDic has been written in C language with Win32 APIs.

  • imedic.c: main window.
  • regword.c: register word dialog.

If you want to compile this program, you will need header and resource files included in the distribution archive.

This program uses the following keywords:

ImmEnumRegisterWord; ImmGetRegisterWordStyle; ImmRegsterWord; ImmUnregisterWord; ImmGetDescription; ImmGetIMEFileName; ImmSetOpenStatus; ImmGetConversionStatus; ImmSetConversionStatus; ImmConfigureIME; RegisterClassEx; CreateWindowEx; CreateStatusWindow; DialogBox; MessageBox; ListView_InsertColumn; ListView_InsertItem; ListView_SetItemText; ListView_SetItem; ListView_DeleteAllItems; ListView_GetItemState; and other basic Win32 APIs.
