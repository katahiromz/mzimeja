How to Change Conversion Mode

To change the IME's mode(s), take the following steps:

  1. Get the current status by ImmGetConversionStatus function.
  2. Calculate the expected mode(s) according to the current status. For Japanese IMEs, generally you should keep the roman/kana flag (IME_CMODE_ROMAN) so as not to change the input mode expected by the user.
  3. Set the new status by ImmSetConversionStatus function.

 

HIMC hIMC;
DWORD dwConvMode; /* conversion mode */
DWORD dwSentMode; /* sentence mode */

/* get the hIMC. */
hIMC = ImmGetContext(hWnd);
if (hIMC != NULL) {
    /* get the current status. */
    ImmGetConversionStatus(hIMC,
        &dwConvMode, &dwSentMode);

    /* let's set the conversion mode as
       half-shape katakana for example. */
    dwConvMode &= ~IME_CMODE_FULLSHAPE;
    dwConvMode |=
        (IME_CMODE_NATIVE | IME_CMODE_KATAKANA);

    /* set the new status to the hIMC. */
    ImmSetConversionStatus(hIMC, dwConvMode, dwSentMode);

    /* now hIMC can be released. */
    ImmReleaseContext(hWnd, hIMC);
}
