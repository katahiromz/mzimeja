Introduction

The IME (Input Method Editor) is a driver to input double-byte (or unicode) characters specific to the Asian languages.

Multiple IMEs on the system
(clicking IME icon on the taskbar)

FE editions of Win95/NT3.51 or higher can hold the multiple IMEs in the system, while the only one IME can be activated on the earlier version of Windows. You can select the active IME for each context (window) at any time.

The system provides the common interface called IMM (Input Method Manager) to control IMEs. It enables applications to give some commands to the IME or get the messages from the IME. IMM APIs are exported by IMM32.DLL in the system folder.
To use IMM APIs, you should #include <imm.h> and link imm32.lib (or load IMM32.DLL dynamically).

 

APPLICATION TYPES

Applications can be categorized into 3 types according to the way to handle IME strings.

An IME Full-aware Application.

In the strict sense, this kind of application will process most IME messages, create IME component windows (the IME window of "IME" class, status window, composition window, candidate window and guideline window) and display IME status by itself, not leaving these tasks to the IME.
But generally, this term often describes an application which will just create a composition window and draw the composition string by itself like Microsoft Word and Windows' Rich Edit Control.

An IME Half-aware Application.

It places the IME component windows at the proper position, in short, supports in-line editing. It just receive the result string which has already converted and determined by the IME as an user input. Most applications using Windows' Edit Control or MFC window class will be included, as well as Notepad.exe.

An IME unaware Application.

It will never handle IME messages and the user can not input IME string in-line on the application window. Normally the IME will display a composition window outside the application window. Today there are few IME unaware applications for Win95/NT4, but some foreign programs for Win16 may belong to this type.
