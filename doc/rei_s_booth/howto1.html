How to Open/Close IME

The ImmSetOpenStatus function will open or close the IME associated with the specified input context. If you want to add auto-IME-open feature to your window, use the function with proper parameters when it gets and looses the input focus.

 

HIMC hIMC = ImmGetContext(hWnd);

if (hIMC != NULL) {
    /* turn IME on. */
    ImmSetOpenStatus(hIMC, TRUE);

    /* or turn IME off:
    ImmSetOpenStatus(hIMC, FALSE); */

    /* don't forget to release the hIMC. */
    ImmReleaseContext(hWnd, hIMC);
}

 

Normally a process has one input context. The IME status, conversion mode, sentence mode and composition string (etc.) are used commonly within the process. If you want to keep these status of a specific window independently of the main process, you can create a new input context by ImmCreateContext and associate it with the target window by ImmAssociateContext.
For more information about multiple input context, see the "Multiui" sample in Win32 SDK.
