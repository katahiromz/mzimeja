How to Disable IME

IME can never be enabled on a window which has no input context. Using this behavior, you can restrict the user input to ASCII characters only. It is useful on an edit control to input a price, percentage, phone number, logon user name, URL or something else.
To remove the input context from a window, associate a NULL context with it by ImmAssociateContext.

 

HIMC hIMC;

/* disable IME. */
hIMC = ImmAssociateContext(hWnd, NULL);

/* do your process. */
...

/* restore the input context on clean-up. */
ImmAssociateContext(hWnd, hIMC);

 

Win98
Windows 98 has a new function called ImmDisableIME, which takes the thread ID as the only parameter. According to the description of the Platform SDK, it disables the IME for the specified thread or the current process.
