Composition Font

Usually the composition string and the candidate list are drawn with the same font as used on the application window. How to support this behavior?

ImmSetCompositionFont can set the font for the IME's composition/candidate window. Just fill a LOGFONT structure with proper values and pass it to the function, the IME will draw its string with the font you've ordered.

If you need to know the current composition font, use ImmGetCompositionFont function.

 

 HIMC hIMC;
 HDC hDC;
 LOGFONT lf;

 if ((hIMC = ImmGetContext(hWnd)) != NULL) {
     hDC = GetDC(hWnd);

     /* fill the LOGFONT structure. */
     memset(&lf, 0, sizeof(lf));
     lf.lfHeight = (24 * GetDeviceCaps(hDC, LOGPIXELSY)) / 72; 
     lf.lfCharSet = SHIFTJIS_CHARSET;
     lstrcpy(lf.lfFaceName, _T("MS Gothic"));

     ReleaseDC(hWnd, hDC);

     /* set the composition font. */
     ImmSetCompositionFont(hIMC, &lf);

     ImmReleaseContext(hWnd, hIMC);
 }
